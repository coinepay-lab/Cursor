什么是 Cursor

Cursor 是一款基于 VSCode 的 AI 编程编辑器，内置先进的大语言模型（LLM），能够帮助开发者更高效地编写代码。

Cursor 的核心功能包括：

AI 代码生成

代码自动补全

代码解释与重构

项目级代码理解

AI 聊天式编程

通过这些功能，开发者可以更快速地完成开发任务。

相比传统编辑器，Cursor 更像是一个 AI 助手 + IDE 的结合体。

如何切换 Cursor 为简体中文

如果你希望使用中文界面，可以按照以下步骤操作：

打开 Cursor 编辑器

使用快捷键打开命令面板

输入以下命令

> language

选择 简体中文

完成后，Cursor 界面将自动切换为中文。
